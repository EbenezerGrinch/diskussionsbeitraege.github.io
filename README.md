# Phrasenschweinfutter

## Falls eine Diskussion langweilig zu werden droht, einfach eine der folgenden Phrasen aus einer passenden Kategorie einwerfen:


### Höflichkeit/Zurückhaltung

Kommen Sie doch bitte wieder, wenn Sie weniger Zeit haben.

Wenn man jemanden mag, bietet man ihm das Du an. Ich finde, das sollten Sie wissen.

Ich habe weder die Zeit, noch die Buntstifte, Ihnen das zu erklären.

Ich lasse Ihnen gern den Vortritt. Dann weiß ich, wovon ich mich fernhalten sollte.

Ihre Meinung ist wertvoll. Vielleicht nicht für dieses Gespräch, aber grundsätzlich schon.


### Realitätsferne/Abstraktion

Sie besitzen einen interessanten Realitätsabstand.

Wenn ich Ihnen Recht gebe, liegen wir beide falsch.

Kompetenz wird nicht in Dezibel gemessen.

Ihre Sichtweise ist so originell, dass sie mit der Realität kaum noch konkurriert.

Schön, dass Sie Probleme von einer Perspektive betrachten, die sonst niemand für relevant hält.

Ihr Verhältnis zur Wirklichkeit ist bemerkenswert flexibel.

Es ist faszinierend, wie Sie Fakten als Vorschläge interpretieren.


### Ironie/Spott

Ich würde mich mit Ihnen sehr gern weiter geistig duellieren, doch ich bemerke, dass sie unbewaffnet sind.

Reden Sie einfach weiter, bis Ihnen etwas einfällt.

Sie sind kognitiv erfrischend unaufdringlich.

Sie glänzen wirklich mit leistungsunabhängigem Selbstbewusstsein.

Sie bringen bemerkenswerte Ruhe in Situationen, die Tempo erfordern.

Sie haben einen ausgeprägten Blick für das Offentsichtliche.

Sie schöpfen auch bei einfachsten Aufgaben Ihr volles Potenzial aus.

Es freut mich, dass Ihre Meinung nicht durch Vorkenntnis getrübt wird.

Sie bringen Abwechslung in eine Gruppe angenehmer Menschen.

Ich bewundere Ihren Mut, sich in Themen einzubringen, bei denen sich andere erst informieren würden.

Sie haben wirklich das Talent, jede Diskussion auf das Niveau eines Kindergeburtstags zu bringen, nur ohne Kuchen.

Ihre Argumentation ist wie ein Kreisverkehr.


### Verabschiedung

Darf ich Ihnen das Tschüß anbieten?

Ich würde mich sehr gern freuen, Sie wiederzusehen.

Es war mir eine Lehre, Sie kennenzulernen.

Ich verabschiede mich, bevor es noch sinnvoll wird.
